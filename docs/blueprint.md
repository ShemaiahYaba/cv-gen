# **App Name**: Vitae Forge

## Core Features:

- Landing Page: A hero section with clear calls to action.
- Authentication Flow: Secure user authentication (sign-up, sign-in).
- Onboarding Flow (Students): Guided onboarding experience tailored for students, providing Skill-Based CV, ATS CV and custom CV options
- Onboarding Flow (Job Seekers): Guided onboarding experience tailored for job seekers, providing ATS CV and custom CV options.
- CV Editor (Skill-Based): CV editor for skill-based CV creation and modification.
- CV Editor (ATS): CV editor optimized for Applicant Tracking Systems.
- CV Editor (Custom): CV editor with advanced customization options for free-form CV design.
- CV Export: Export CVs in PDF and DOCX formats.
- Dashboard: User dashboard to manage and track created CVs.

## Style Guidelines:

- Primary color: Deep black (#000000) for a sophisticated feel.
- Background color: Off-white (#F0F0F0) to provide a clean and modern contrast.
- Accent color: Dark gray (#333333) to highlight key UI elements and CTAs.
- Body font: 'Montserrat' (sans-serif) for readability and a professional look.
- Headline font: 'Poppins' (sans-serif) for headings, titles, and emphasis.
- Note: currently only Google Fonts are supported.
- Use simple, line-based icons for a minimalist aesthetic.
- Clean, grid-based layout with ample whitespace for easy scanning and navigation.
- Subtle SVG animations on the landing page to capture user attention without distraction.
